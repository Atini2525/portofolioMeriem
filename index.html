Reflection.

• Outline the major design decisions you made for your visuals, explain how you got there and why you picked them.

The main idea was to design a map which shows the different rate of happiness on a legend color. The first set of data that I have investigated was from Kaggle. Kaggle is a subsidiary of Google LLC, is an online community of data scientists and machine learning practitioners. After, a little bit of research I have encountered this website “Our World in Data”, which is a scientific online publication that focuses on large global problems such as poverty, disease, hunger, climate change, war, existential risks, and inequality. I have decided then to use “Our World in Data” as my main source of information since it has a larger set of data.
Once the exploratory project was completed, the next step was to interpolate the happiness data with another set variables. The variable that was added for this study was the fertility rate from “Our World in Data”. The purpose is to display the relationship between the fertility rate and the happiness rate. The visualization of these variables would help to assess if the fertility rate would affect positively or negatively the happiness rate. Scatter plots were used for the comparison. Figure 01 is a scatter plot, that represents the happiness rate (X Axis) vs countries (Y Axis). The order arrangement was made such as the countries with the lowest happiness rate would be in the bottom, and the countries with the highest happiness rate would be in the top. Figure 02 is a scatter plot that represent the fertility rate vs countries. the order arrangement of these countries was made to match the Y axis in figure 01.
The line charts represent the changes of both fertility rate and happiness rate for each country during the last decade. These charts show that they can be reliable to draft serious conclusions and relationship between these two data sets.

How did your final project differ from the prospectus you outlined?

My final project differs significantly from the prospectus design. I have added a map which was not included in my preliminary project prospectus. The reason why I have added it it’s because it can be helpful to visualize and display the happiness index per country. Similarly, two-line charts, and three scatter plots were added on the final project. The purpose of adding two-line chart per country is to provide more explanation and more insight that can be used to draft a better conclusion. Overall, I am satisfied with the work I have performed, and I wish that I would have had more time to do a more detailed work.

What were your major challenges? How would you mitigate them with your next project?

I have spent lot of time and energy working on the map. It was one of my biggest challenges. I had to learn how to connect the Json file to CSV file in order to match the data from each file. I had to create the ID of the country in my CSV data in order to compare it. Also, I had to set two loop functions. The first one is to loop in the Json file and the second one is to loop in the CSV file. The most challenging part was to add a zoom and a color scale.
Also, one of the challenges that I have encountered was to set up a tooltip on the different maps. The hardest part is to set up the boundaries of each country. So, when we move the cursor over a specific country, we can get the data as long as the cursor is within the boundary of that specific country.
Last but not least, I had to learn how to clean the data set. Cleaning the data is a really important step that helps displaying the data in a neat way. Personally, it took me more than a day to find the correct data and clean it.

If you had more time and resources, what would you add/remove/change about your projects?

If I had more time and more resources, I would have tried to investigate the difference of happiness rate and fertility rate since the 1950’s. However, I had faced difficulties to find the happiness rate before the 2000’s. Unfortunately, I had to near down my research to only the last decades. Therefore, we cannot fully explain the evolution of both fertility and happiness. I also wanted to see the effects of income on the happiness rate. In the 1950’s the income was lower, but the fertility was higher, if I had more data, I would have drafted a more detailed conclusion. If I had more time I would have designed one more scatter plot where the x scale would be the rate of happiness and y scale will be the rate of fertility , which will plot the difference of the country comparing to these two data. Moreover, I would have added more details and more analysis to support my conclusion.
